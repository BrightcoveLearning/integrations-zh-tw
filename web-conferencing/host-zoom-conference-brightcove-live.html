    在布萊特灣現場舉辦一場變焦會議

    在本主題中，您將學習如何串流 Brightcove 直播的 Zoom 會議或網路研討會。

    簡介

    Zoom 視訊通訊是一家技術公司，提供雲端式、點對點軟體平台，適用於電話會議、遠端通勤和遠距教育。若要接觸更多觀眾，您可以利用 Brightcove Live 串流您的會議，讓其他人觀看。

    步驟

    請依照下列步驟舉辦會議或網路研討會，參加 Brightcove 直播：

    1. 會議

      對於 Zoom 會議，請執行下列動作：

      1. 瀏覽至您的縮放會議設定，然後捲動至「會議中」(進階區段)。
      2. 按一下切換開關，啟用 [ 允許即時串流會議 ] 選項。如果您沒有看到此選項，請洽詢您的縮放帳戶管理員，了解如何將此功能新增至您的帳戶。

      3. 選取「自訂即時串流服務」，然後選擇性地在方塊中輸入文字備註。

      網路研討會

      對於「縮放」網路研討會，請執行下列動作：

      1. 瀏覽至您的縮放網路研討會設定，然後捲動至 [ 在網路研討會設定] 區段。
      2. 啟用 [ 允許主機即時串流其網路研討會 ] 選項。

      3. 選取 [ 自訂串流服務 ]，然後儲存。

    2. 在Video Cloud Studio中，創建一個使用媒體模塊的現場直播。

      如果您想在廣播完成後儲存，請務必勾選「完成時將事件轉換為視訊資產」核取方塊！

    3. 當您建立即時事件時，您將可以存取串流 URL 和串流金鑰。

      • 串流 URL-串流端點 URL (您可以在 Brightcove 即時控制室找到端點 URL
      • 流密鑰-流名稱（在 Brightcove 中，流名稱始終是「活著」）
    4. 在您的網站/內部網路頁面中發佈並嵌入此 Live 活動的播放程式，並記下網頁 URL。
    5. 會議

      如果是 Zoom 會議，請瀏覽至排定的會議清單，然後選取您要串流的會議。

      網路研討會

      如果是縮放網路研討會，請返回「網路研討會」畫面，並安排網路研討會。選取「即時串流」標籤。

    6. 捲動至「即時串流」區段
    7. 選取 [設定即時串流設定] 連結。您將在這裡輸入串流網址、串流索引鍵以及代管播放器的頁面的 URL。
    8. 從縮放通話中，選取「更多」按鈕，然後選取「在自訂即時串流服務上即時直播服務」選項。這將開始直播。
