Adobe-AEM-Brightcove 連接器：安裝

本主題涵蓋了Adobe Experience Manager（AEM）的Brightcove連接器的安裝。

簡介

Abe-AEM-BrightCove 連接器可讓您管理 AEM 中的 Brightcove 視訊雲端視訊和播放器，並輕鬆將影片嵌入 AEM 頁面中。

請注意，本指南使用 AEM 6.3。該連接器也適用於 AEM 6.2、6.4 和 6.5，步驟類似。

開始使用

如果您還沒有AEM開發人員 例如，您可以使用AEM Quickstart .jar文件輕鬆創建一個：

java  -Xmx1400M -jar AEM_6.3_Quickstart.jar

如果使用此快速入門，則默認情況下，您的AEM實例將託管在：http：// localhost：4502。以下各節將採用該默認位置，因此，如果您使用其他位置或端口，則需要調整步驟中顯示的URL，或者改用AEM導航菜單。

安裝連接器

以下步驟將引導您完成使用經典界面安裝連接器的過程。

  • 去https://github.com/brightcove/Adobe-AEM-Brightcove-Connector/releases/並下載 您的AEM版本的連接器（Brightcove-all ... zip文件）。
  • 啟動您的AEM開發實例。
  • 登錄後，轉到CRX軟件包管理器（http：// localhost：4502 / crx / packmgr / index.jsp）
  • 點擊上傳按鈕：
  • 在“上載"對話框中，瀏覽到該連接器的zip文件，然後單擊“確定"將其上載。
  • 當程序包出現在列表中時，單擊“安裝"進行安裝：
  • 轉到控制台捆綁包頁面（http：// localhost：4502 / system / console / bundles）來驗證該軟件包是否處於活動狀態：

完成這些步驟後，連接器已成功安裝，您應繼續執行以下操作：配置。
