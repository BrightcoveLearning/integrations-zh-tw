WordPress-Brightcove 連接器：安裝

本主題涵蓋 WordPress 專用的布萊特灣連接器的安裝。

簡介

WordPress 布萊特灣視頻連接器允許您管理亮灣視頻雲視頻和 WordPress 中的播放器，並輕鬆地在 WordPress 頁面中嵌入視頻。

安裝步驟

  1. 下載WordPress Brightcove Video Connect插件
  2. 解壓縮文件並移動/brightcove-video-connect至/wordpress/wp-content/plugins
  3. 透過 WordPress 中的外掛程式選單啟用亮灣視訊連線

如果在側邊欄中可見Brightcove插件，則可以移至配置。
