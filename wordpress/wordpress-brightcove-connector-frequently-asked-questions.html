WordPress-Brightcove 連接器：常見問題

本主題包含有關字詞壓縮連接器的常見問題集。

我可以在無法公開訪問的 WordPress 安裝上運行插件嗎？

是的，無論它是否公開或不需要回電到您的網站的功能，如狀態更新，上傳視頻等將無法使用。

 是否有針對插件/主題開發人員的過濾器？ 

亮麗的 cove_account_動作 = [編輯，刪除];//操作布萊特灣來源時可以使用哪些動作。

這可以在多站點上工作嗎？

是的，它會的。

我可以使用一個以上的布萊特灣帳戶嗎？

可以，如果需要，您可以從許多 Brightcove 帳戶新增來源。

我可以為使用者設定不同的權限嗎？

是的您可以在下面找到用戶權限矩陣這個文件。要更改用戶角色，我們建議使用WordPress add_cap函數： https://codex.wordpress.org/add_cap

同步如何運作？

該插件只是直接從 Brightcove API 提取信息進行顯示，並且不會在本地同步視頻。

如何增加上傳檔案大小上限？

最大文件大小由您的網絡服務器和 PHP 配置確定。你需要在你的 php.ini 中設置上傳的值和文件大小。php_ini____ADVD_ 文件（）可以幫助你找到你的 PHP.ini 所在的位置。

如何啟用變更追蹤功能，以便查看誰更新影片，以及他們何時更新影片？

創建一個類型為 '文本' 的內部名稱' _change_歷史' 的自定義字段。每當影片更新時，使用者名稱和目前時間都會新增至此欄位中記錄的變更清單中。

http://nginx.org/en/docs/http/ngx_http_core_module.html#client_max_body_size (客戶端最大尺寸)
對於阿帕奇：
