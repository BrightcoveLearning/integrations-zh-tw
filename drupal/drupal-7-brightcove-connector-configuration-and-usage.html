Drupal 7-Brightcove 連接器：組態與使用

簡介

Drupal Brightcove 視頻連接器允許您管理布萊特灣視頻雲視頻和 Drupal 中的播放器，並輕鬆地在 Drupal 頁面中嵌入視頻。本頁詳細介紹如何配置 Drupal 7 中的布里特灣模塊。對於Drupal 8，請參閱Drupal-Brightcove連接器：配置。

Drupal 安裝是一個複雜的過程，很容易出錯。請確定您的內容已備份，然後再繼續進行。

連線到布萊特灣帳戶

  1. 要連接您的 Brightcove 帳戶，請通過單擊導航到 Brightcove API 頁面 組態，然後向下滾動到 Brightcove 設置 選項下 媒體。 這將帶您到布萊特灣設置頁面。要添加新的 Brightcove API 客戶端，請單擊 API 客戶端 選項卡，然後選擇 添加 API 客戶端。 這將帶您到要在 API 客戶端中添加的表單。
  2. 標籤可以是任何東西，但使用與 Brightcove 上的帳戶名稱相同的名稱將使其更容易識別。
  3. 要查找帳戶ID，請查看Brightcove Studio的左上角 儀錶盤。
  4. 要獲取客戶端 ID 和客戶端密碼，你將不得不註冊一個應用程序。在 Brightcove Studio 中，導航至 API 驗證 並點擊 添加新應用程序。
  5. 填寫名稱和簡短描述。
  6. 選取帳戶。下圖為所有必要的權限。
  7. 點擊儲存。
  8. 會出現用戶端識別碼和用戶端密碼。 確保將Client Secret複製並保存到安全位置。您將無法再次訪問它。
  9. 填寫所有 ID 和密鑰。 點擊儲存。
  10. 如果您遇到問題，請確定已設定日期和時間，然後再試一次。

添加布萊特灣視頻場

要添加 Brightcove 視頻到您的 Drupal 帖子，首先添加一個視頻字段到新的內容頁面。要做到這一點，導航到「結構」，然後單擊「內容類型」。「

從您希望新增 Brightcove 影片的內容類型中選擇「管理欄位」。這個過程可以針對每個內容類型重複。

按一下「新增欄位」。「標記它，然後選擇「Brightcove」作為場地類型，並選擇「視頻（上傳/瀏覽）」小部件。點擊儲存。

您的視訊欄位現在應該會出現。

張貼和管理視訊

與 Drupal 8 不同，Drupal 7 連接器沒有一個單獨的內容管理系統，所以大多數內容管理應該在布賴特灣工作室網站上完成。不過，在新的內容頁面中，您可以按一下瀏覽以檢視所有影像，也可以上傳新的影像。

有兩種方法可以將現有視訊附加至貼文。如果您知道影片的標題，請在 Brightcove 欄位中輸入影片，並允許自動完成功能找到影片。您也可以點擊瀏覽按鈕來查看和選擇所有視頻。

點擊上傳按鈕將打開上傳表單，從而允許您直接從 Drupal 添加新的視頻。發布帖子，你的視頻應該會出現。
