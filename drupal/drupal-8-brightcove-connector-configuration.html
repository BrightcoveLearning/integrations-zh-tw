Drupal 8-Brightcove 連接器：配置

簡介

Drupal Brightcove 視頻連接器允許您管理布萊特灣視頻雲視頻和 Drupal 中的播放器，並輕鬆地在 Drupal 頁面中嵌入視頻。請注意，這適用於 Drupal 8。對於Drupal 7，請參閱Drupal 7指令。

Drupal 安裝是一個複雜的過程，很容易出錯。請確定您的內容已備份，然後再繼續進行。

連線到布萊特灣帳戶

  1. 要連接您的 Brightcove 帳戶，請通過單擊導航到 Brightcove API 頁面 組態，然後向下滾動到 Brightcove API 帳戶 選項下 媒體。 這將帶您到布萊特灣 API 客戶端頁面。要添加新的 Brightcove API 客戶端，請單擊 添加 Brightcove API 客戶端。
  2. 標籤可以是任何東西，但使用與 Brightcove 上的帳戶名稱相同的名稱將使其更容易識別。
  3. 要獲取客戶端 ID 和客戶端密碼，你將不得不註冊一個應用程序。在 Brightcove Studio 中，導航至 API 驗證 並點擊 添加新應用程序。
  4. 填寫名稱和簡短描述。
  5. 選取帳戶。下圖為所有必要的權限。
  6. 點擊儲存。
  7. 會出現用戶端識別碼和用戶端密碼。 確保將Client Secret複製並保存到安全位置。您將無法再次訪問它。
  8. 要查找帳戶 ID，請轉到管理頁面的帳戶信息部分：
  9. 填寫所有 ID 和 Keys 並選擇 默認播放器。
  10. 點擊 保存。API 用戶端現在應該會出現在清單中。

同步

最初，當您導航到 新內容 頁面並點擊 Brightcove 視頻，不會有任何內容，但會提供添加視頻的選項。這是因為首先，您需要同步資料。初始同步後，Drupal 的 chron 將所有內容保持在最新狀態。如需手動同步，請單擊 報告 頂部的按鈕，然後單擊 Brightcove 狀態概覽。

在 Brightcove 狀態概覽 頁，點擊 全部同步。

再次導覽至 Brightcove 影片頁面，您的影片應該可供使用。

設定權限

您現在可以檢視和張貼視訊，但是如果非系統管理員使用者要與視訊互動，則必須設定權限。要設置這些權限，首先通過單擊導航到權限頁面 人們 進而 權限。

向下捲動以尋找布萊特灣權限。根據您的需求啟用和停用。Brightcove 權限會自動設定為僅限管理員，因此如果您希望其他人觀看影片，您必須授與這些權限。

現在您可以繼續內容管理或者嵌入內容。
