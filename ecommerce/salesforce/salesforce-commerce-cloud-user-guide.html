多語言支援

該擴展為業務管理器界面提供多種 UI 語言支持。注意：此版本的擴充功能支援英文和法文的 UI 語言。

加入視訊

以下是添加視頻的步驟。

  1. 導航商家工具-> Brightcove->添加視頻
  2. 選擇語從菜單中選擇（默認將是為站點選擇的默認語言環境）：
  3. 點擊選擇視頻字段旁邊的選項。
  4. 在顯示的對話方塊中，選取視訊檔案：
  5. 點擊選擇縮略圖字段旁邊的選項，將出現一個類似的彈出對話框，您可以在其中選擇縮略圖文件。
  6. 點擊選擇海報字段旁邊的選項，將出現一個類似的彈出對話框，您可以在其中選擇海報文件。
  7. 提交表單以啟動擷取程序：
  8. 提交後，驗證表單上的成功消息：
  9. 您可以登入 Brightcove Studio 並導覽至媒體模組，確認視訊中是否已成功攝入：
  10. 您也可以按一下影片以顯示詳細資料，以驗證地區設定代碼：

嵌入視訊

內嵌視訊步驟可讓您取得程式碼，以便在任何 ISML 頁面上呈現視訊。

  1. 導航商家工具-> Brightcove->嵌入視頻。
  2. 選擇所需的視頻和播放器，然後單擊取得驗證碼：
  3. 將出現一個彈出對話框，其中包含代碼-選擇代碼塊，它將被複製到剪貼板：
  4. 將程式碼貼到內容資產、內容位置或任何 ISML 範本中，以轉譯視訊。

請注意，videofile , videoposter和videothumbnail必須在可公開訪問的路徑上可用。

  2. 選取 .csv 檔案。
  3. 按一下提交：
