設定自訂網站偏好設定

  1. 選擇一個地點從下拉菜單中：
  2. 導航商家工具->網站偏好設置->自定義偏好設置。
  3. 選擇布萊頓灣。
  4. 新增下列值：
    • 布萊特灣帳號 ID
    • 布萊特灣用戶端識別碼
    • 布萊特灣客戶機密
    • 網站目錄
    看管理 API 身份驗證憑據有關如何生成客戶端 ID 和密碼的信息。您的認證將需要這些權限：
  5. 新增布萊特灣槽限制 (1-11)。預設值為 11。
    • 主產品插槽（掌握和m1-m10）
    • 各種產品的插槽（ v1-v11）
    • 類別插槽（ c1-c11）

    插槽限制為 5 將提供下列插槽進行設定：

    • 主產品（掌握和m1-m4）
    • 各種產品（ v1-v5）
    • 類別（ c1-c5）

在 ISML 檔案中包含布萊特灣視訊渲染

SFRA墨盒–app_brightcove

  1. 以下 ISML 檔案支援視訊轉譯：
    • productDetails.isml
    • catLanding.isml
    • searchResultsNoDecorator.isml
  2. 包括modules.isml： 
    <isinclude template="modules">
  3. 包含來自的標籤modules.isml渲染視頻：
    • 主產品視頻<isM1> -這將渲染分配給11投幣口
    • 多種產品視頻<isV1> -這將渲染分配給v1投幣口
    • 分類視頻<isC1> -這將渲染分配給C1投幣口

SiteGenesis濾芯–int_brightcove

  1. 以下 ISML 檔案支援視訊轉譯：
    • productcontents.isml
    • producttopcontent.isml
    • categoryproducthits.isml
    • catlanding.isml
  2. 包括modules.isml： 
    <isinclude template="modules">
  3. 包含來自的標籤modules.isml渲染視頻：
    • 主產品視頻<isM1> -這將渲染分配給11投幣口
    • 多種產品視頻<isV1> -這將渲染分配給v1投幣口
    • 分類視頻<isC1> -這將渲染分配給C1投幣口
