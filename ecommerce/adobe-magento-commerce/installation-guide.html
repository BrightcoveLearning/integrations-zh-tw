準備安裝

萃取檔案和資料夾並複製到工作資料夾

  1. 建立資料夾[Magento_Root]/app/code/Brightcove/Upload
  2. 下載歸檔檔案 (程式碼檔案)。
  3. 解壓縮檔案/資料夾
  4. 在步驟 1 中將解壓縮檔案的內容拖放/移動到目錄

啟用擴充功能和外掛程

啟用下列項目：

  • php bin/magento module:enable Brigtcove_Upload
  • php bin/magento setup:upgrade
  • php bin/magento cache:clean
  • php bin/magento setup:static-content:deploy
  • php bin/magento cache:flush

設定布萊特灣模組認證

若要完成此部分，您必須取得延伸模組使用之 API 的用戶端憑證。有關如何創建憑據的詳細信息，請參見管理API身份驗證憑據。憑證所需的最低權限如下所示：

  1. 導航存儲->配置-> Brightcove（左側標籤）-> Brightcove視頻配置
  2. 輸入布萊特灣配置的詳細資料：
    • 客戶編號：REST 用戶端/使用者 ID
    • 客戶機密：REST 客戶端密碼/密碼。
    • API基本網址：
    • ID：布萊特灣商店 ID
    • 類別視頻廣告位：將視訊槽數設定為「品類欄」的步驟
    • 主視頻插槽：將視訊插槽數目設定為主欄
    • 變體視訊插槽：將視訊插槽數目設定為變體欄
    • 電子郵件 ID：傳送與大量上傳狀態相關的電子郵件
  3. 正確提供所有詳細信息後，保存配置頁面。現在，用戶應該可以訪問布萊特灣自定義模塊。

停用擴充功能

如果您需要停用 Brightcove 擴充功能，請使用下列命令：

  1. php bin/magento module:disable Brightcove_Upload
  2. php bin/magento setup:upgrade
  3. php bin/magento cache:clean
  4. php bin/magento setup:static-content:deploy
  5. php bin/magento cache:flush
